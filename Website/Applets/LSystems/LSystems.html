Part 4 of 4: Iterated Function Systems

The Collage Theorem is a means of creating a desired image through tiling of similar contractive images together. By mapping where points on the final image end up, you are able to create a system of equations where the variables are the elements in the matrix definition of the affine transform. You only need to use 3 vertices for any shape as the last one will follow in order to retain co-linearity. Each set of points has a certain probability, usually 0.33, which indicates which set is selected to plot - this is the concept of the Chaos Game.
